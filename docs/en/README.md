# OpenSource for Humans

OpenSource for Humans is an open source proposal for connecting people with nearby, trustworthy, and up-to-date social support services.

The project vision is to organize access to social help as open public infrastructure: built by people, for people, with technology serving human dignity.

English is the primary language for the project documentation. Portuguese translations are available in [../pt-BR](../pt-BR).

## Core Rule

People seeking help are never the product.

This means the project must not charge people who are looking for help, sell personal data, prioritize essential services because of payment, or use AI to make final decisions about rights, health, safety, housing, emergency support, or access to care.

At this initial stage, personal data from people seeking help and general visitors must not be stored server-side. If user-related data is ever needed for statistics, it must be anonymized and aggregated before storage so the person cannot be identified.
